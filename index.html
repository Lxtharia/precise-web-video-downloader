Life can be so simple

Select audio option:

Controls:

  • Paste Youtube Url
  • Let is play, navigate with mouse, whatever
  • Pause/Unpause with SPACE
  • Navigate frame by frame with . (DOT) and , (COMMA)
  • Navigate second by second with the Arrow Keys (Note: If the player is focused, it will skip by 5 or 10 seconds and won't catch Start/End key bindings)
  • Set Start and End Time with S and E
  • Generate yt-dlp command and execute it in on the command line, it will download it in the current folder by default
  • If the time is displayed 12:00 AM/PM, instead of 00:00, its cause you cant change that in html/css so it's your locale settings. I find it very annoying.
